## 仕組み 🎅

1. 提供されている呪文で必要なライブラリをインストールします。
2. `MY_OPENAI_API_KEY` を設定して、LLM（Language Learning Model）に接続します。
3. プロンプトを生成し、Retrieverが求める回答を取り出させましょう。
